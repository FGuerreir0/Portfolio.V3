# Fábio Guerreiro - Portfolio

Modern, responsive portfolio website built with React, Vite, and cutting-edge web technologies.

![Portfolio Preview](https://img.shields.io/badge/React-18.2-blue)
![Vite](https://img.shields.io/badge/Vite-5.0-purple)
![License](https://img.shields.io/badge/license-MIT-green)


## 🛠️ Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** CSS3 with CSS Variables
- **Deployment:** Netlify

## 🌐 Deployment

### Netlify

1. Connect your GitHub repository to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `dist`
4. Deploy!
